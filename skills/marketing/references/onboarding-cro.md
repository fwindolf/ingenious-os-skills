# Onboarding CRO

Optimizing post-signup activation and time-to-value for ingeniOS.

## Key Concept: Time-to-Value

Remove every step between signup and the "aha moment". For ingeniOS, likely activation events:
- First Rechnung created (invoicing value)
- First Stunden logged (time tracking value)
- First Kunde added (organization value)

## Onboarding Flow Options

- **Product-first**: Drop into product immediately, guide contextually (best if product is simple)
- **Guided setup**: Ask a few questions, personalize (trade, company size, priorities)
- **Value-first**: Pre-populate with demo data so user sees a working system immediately

For Handwerker (non-tech, time-pressed): **value-first + guided** — show demo Betrieb, then guide through making it theirs.

## Onboarding Checklist

- 3-7 items max
- Order by value impact: highest-value action first
- Show progress %
- Celebrate completion
- Dismiss option (don't trap users)

Example:
1. ✅ Betrieb einrichten (Name, Adresse, Logo)
2. Ersten Kunden anlegen
3. Erste Rechnung schreiben
4. Mitarbeiter einladen
5. Zeiterfassung starten

## Empty States

Every empty state should:
- Explain what this area does
- Show what it looks like with data
- Clear primary action CTA
- Optional: pre-populate with example data

## Email + In-App Coordination

Trigger-based emails reinforcing in-app progress:
- **Welcome** (immediate): "Willkommen! So geht's los"
- **Incomplete setup** (+24h): "Noch 2 Schritte bis Ihr Betrieb digital läuft"
- **Activation achieved**: "Ihre erste Rechnung — so geht's weiter"
- **Feature discovery** (+3-5 days): "Wussten Sie schon? Zeiterfassung direkt vom Handy"

## Stalled Users

Detect: X days inactive, incomplete setup
- Email sequence: value reminder → address blockers → offer help
- In-app recovery prompt on return
- Personal outreach for early customers (Flo email)

## Metrics

- Activation rate (% who complete key action)
- Time to activation
- Onboarding checklist completion rate
- Day 1 / Day 7 / Day 30 retention
- Drop-off per onboarding step
