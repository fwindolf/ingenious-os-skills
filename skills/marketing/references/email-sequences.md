# Email Sequences

Drip campaigns, welcome sequences, and nurture flows for ingeniOS.

## Core Principles

- One email, one job, one CTA
- Value before ask
- Fewer better emails > more mediocre ones
- German language, Handwerk tone (direct, practical, no corporate speak)

## Email Structure

Hook (first line) → Context → Value → CTA → Sign-off

**Formatting**: Short paragraphs (1-3 sentences), white space, bullets sparingly, bold sparingly, mobile-first. Length: 50-125 words (transactional), 150-300 (educational).

**Subject lines**: Clear > Clever, 40-60 chars. Patterns: Question, How-to, Number, Direct.

## Sequence Types

### Welcome Sequence (5-7 emails, 12-14 days)
1. **Welcome + first step** (immediate): "Willkommen — so starten Sie in 5 Minuten"
2. **Quick win** (+1 day): Help them do ONE valuable thing
3. **Story** (+3 days): Why we built ingeniOS (Handwerker problem)
4. **Social proof** (+5 days): Another Betrieb's experience
5. **Overcome objection** (+7 days): Address "hab keine Zeit" or "mein System funktioniert"
6. **Core feature deep dive** (+10 days): The feature most relevant to their trade
7. **Conversion** (+14 days): Clear ask to stay / upgrade

### Lead Nurture (6-8 emails, 2-3 weeks)
For leads who downloaded a lead magnet or signed up for newsletter:
1. Deliver the resource
2. Expand on the topic
3. Problem deep-dive (use numbers)
4. How ingeniOS helps
5. Case study / social proof
6. Direct offer / invite to try

### Onboarding (5-7 emails, 14 days)
Coordinate with in-app onboarding:
1. Welcome + first step
2. Help if stuck (+24h if no activation)
3. Feature highlight
4. Success story
5. Check-in (+7 days)
6. Advanced tip
7. Upgrade prompt (if on trial)

### Re-engagement (3-4 emails, 2 weeks)
For inactive users:
1. "Wir vermissen Sie" — check-in
2. Value reminder — what they're missing
3. Incentive (extended trial, help call)
4. Last chance — clear deadline

## CTA Guidelines

- Buttons for primary CTA
- Text links for secondary
- One clear primary CTA per email
- Action + outcome: "Erste Rechnung schreiben" not "Jetzt einloggen"

## Metrics

- Open rate: target 30-50% (warm list)
- Click rate: target 2-5%
- Unsubscribe rate: <0.5% per email
- Sequence completion rate
- Conversion to activation/paid
