# Page CRO

Optimizing marketing pages for conversions.

## Analysis Framework (Priority Order)

1. **Value Proposition Clarity** (highest impact)
   - 5-second test: can you understand what + why care?
   - Specific > vague, customer language

2. **Headline Effectiveness**
   - Outcome-focused, specific with numbers/timeframes
   - Match traffic source

3. **CTA Placement & Copy**
   - Visible without scrolling
   - Button copy = value not action ("Kostenlos starten" not "Absenden")
   - One clear primary action
   - Repeated at decision points

4. **Visual Hierarchy / Scannability**
   - Clear reading path
   - Key info scannable in 10 seconds
   - Mobile layout works

5. **Trust Signals**
   - Testimonials (specific, attributed, photos)
   - Logos, review scores, security badges
   - Placed near CTAs

6. **Objection Handling**
   - FAQ format, address: price, complexity, data safety, switching
   - "Ist das DSGVO-konform?" → German hosting, Frankfurt
   - "Hab keine Zeit für sowas" → Setup in 15 Minuten

7. **Friction Points**
   - Every click/field = potential drop-off
   - Remove unnecessary steps

## Experiment Ideas

**Homepage**: Hero headline A/B, CTA copy, social proof placement, video vs static
**Pricing**: Tier naming, annual discount prominence, feature comparison depth, FAQ content
**Landing Pages**: Headline formula, form length, trust signal placement, urgency elements
**Feature Pages**: Demo video vs screenshots, benefit framing, comparison to status quo

## Output Format

- **Quick Wins**: Implement now (copy fixes, CTA improvements)
- **High-Impact Changes**: Prioritize (layout, messaging, trust signals)
- **Test Ideas**: Hypotheses with expected impact
- **Copy Alternatives**: 2-3 options for headlines/CTAs
