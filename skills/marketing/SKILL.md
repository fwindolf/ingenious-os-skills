# Marketing

Marketing skills for ingeniOS — the operating system for German Handwerk trade contractors.

## Context

Read CLAUDE.md at workspace root for conventions. For product positioning, ICP, brand voice, and customer language, read `knowledge/product-marketing-context.md`. This is essential context for any marketing work.

## Routing

Identify the domain and read the corresponding reference:

**Copywriting** — writing or improving page copy (homepage, landing, pricing, feature, about)
- → Read `references/copywriting.md`

**Content Strategy** — planning what content to create, topic clusters, editorial calendar
- → Read `references/content-strategy.md`

**Competitor Pages** — creating "vs" or "alternative to" comparison pages
- → Read `references/competitor-pages.md`

**SEO** — auditing or improving search engine optimization
- → Read `references/seo.md`

**Launch Strategy** — planning product launches, feature releases, Product Hunt
- → Read `references/launch.md`

**Lead Magnets** — creating gated content for email capture
- → Read `references/lead-magnets.md`

**Pricing Strategy** — tier structure, pricing research, pricing page
- → Read `references/pricing.md`

**Page CRO** — optimizing any marketing page for conversions
- → Read `references/page-cro.md`

**Signup Flow CRO** — optimizing registration and trial activation
- → Read `references/signup-cro.md`

**Onboarding CRO** — optimizing post-signup activation and time-to-value
- → Read `references/onboarding-cro.md`

**Email Sequences** — drip campaigns, welcome sequences, nurture flows
- → Read `references/email-sequences.md`

Some requests span multiple domains (e.g., "launch the pricing page" = launch + pricing + copywriting). Read all relevant references.

## General Principles

- Always read `knowledge/product-marketing-context.md` first for brand voice, ICP, and positioning
- German language for all customer-facing content (use customer language from context doc)
- No corporate speak — direct, simple, like talking to a Meister
- Mobile-first — contractors read on phones at Baustellen
- Specificity wins — "Rechnungen in 3 Minuten" beats "schneller Rechnungen schreiben"
- Cite real problems with numbers where possible
- Create marketing tasks in the backlog (label: `gtm`)
