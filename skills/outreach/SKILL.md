# Outreach

Find prospects and run outreach campaigns for ingeniOS — the operating system for German Handwerk trade contractors.

## Context

Read CLAUDE.md at workspace root for conventions. CRM lives at `crm.yml` in workspace root. For product positioning and ICP, read `knowledge/product-marketing-context.md`.

## Routing

Identify which domain the request falls into:

**Prospect Research** — finding and qualifying companies
- "find SHK companies", "research this company", "build a prospect list", "add to CRM"
- → Read `references/prospect-research.md`

**Campaign Execution** — planning and sending outreach
- "create a campaign", "draft outreach emails", "brainstorm angles", "send to prospects"
- → Read `references/campaign.md`

Some requests span both (e.g., "find SHK companies in Berlin and email them about Zeiterfassung"). Start with prospect research, then move to campaign.

## General Principles

- Never send without Flo's explicit approval
- Always personalize — no mail merge blasts
- Research before messaging — angles must be grounded in real problems
- Track everything in CRM — every email sent = interaction recorded
- German language for all prospect-facing content
- Use `knowledge/product-marketing-context.md` for brand voice and customer language
